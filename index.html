ASX Financial News Sentiment Analyser

Scraped live financial news for five ASX-listed companies (CBA, BHP, NAB, WES, ANZ) and scored headlines using FinBERT to build a continuous sentiment signal correlated against stock price movements.

  • Pearson correlation of r = +0.34 at the 1-day forward return window, directionally consistent with the hypothesis
  • FinBERT (BERT pre-trained on financial text) produces scores from −1.0 (bearish) to +1.0 (bullish), joined to 90-day adjusted close prices via yfinance
  • Interactive Streamlit dashboard deployed via Docker on HuggingFace Spaces, live and publicly accessible
Python FinBERT HuggingFace Transformers PyTorch NLP SQLite Pandas Plotly Streamlit Docker yfinance

Brisbane Housing Affordability Analysis

Combined Queensland Spatial Catalogue geospatial data, ABS population projections, and residential land supply data across 33 Brisbane SA2 regions to map dwelling shortages.

  • Random Forest Classifier achieved 83% accuracy labelling regions as Low/Medium/High supply
  • Random Forest Regressor (R²=0.489) estimated dwelling yield per region through 2046
  • Identified specific suburbs where housing demand is most likely to outpace supply, directly actionable for urban planners
